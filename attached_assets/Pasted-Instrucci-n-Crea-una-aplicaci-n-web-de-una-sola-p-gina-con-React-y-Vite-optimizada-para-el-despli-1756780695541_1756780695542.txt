Instrucción:

Crea una aplicación web de una sola página con React y Vite, optimizada para el despliegue en la plataforma Vercel. La aplicación se llamará "TIENDA DE PRODUCTOS STAF" y será completamente responsive.

Estructura y Configuración:

Crea un proyecto de React usando Vite.

La estructura debe incluir carpetas para componentes, activos (assets), y estilos (styles).

Utiliza CSS modular para el diseño y el package.json debe estar configurado para un despliegue sin problemas en Vercel.

Página Principal (Home) y Contenido:

Diseña la página de inicio para mostrar productos organizados por categorías: Ropa, Calzado, Cosméticos, Aseo Personal y Salud.

Cada categoría debe tener un título. Dentro de cada sección, muestra una cuadrícula de productos con su nombre, imagen y precio en dólares.

La IA debe buscar y descargar imágenes representativas de los productos desde internet para incluirlas directamente en la carpeta de activos del proyecto.

Consideraciones Clave para el Despliegue:

El proyecto debe ser fácilmente descargable y compatible con los comandos de build de Vite para un despliegue automático en Vercel. No se deben requerir configuraciones adicionales en Vercel.
